# Project

Micro frontend using Svelte and Bun

## Requirements

* Node 18
* [BunSh](https://bun.sh/docs) 1.0

## Getting started

```bash
# install dependencies
bun install

# run application in develop mode
bun run dev

# build project
bun run build

# run project builded
bun run preview
```
